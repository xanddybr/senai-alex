Aula 01 · Módulo Específico Profissional

Vamos pensar como um programador!

Cada exercício você fará duas vezes: primeiro como fluxograma no Draw.io para visualizar a lógica, depois como código JavaScript no VSCode para ver o computador executar. Mesma ideia, duas linguagens.

Curso Programador Front-End
Unidade Curricular Lógica de Programação · 32h
Professor Alexandre Santana
Total de exercícios 30 · cerca de 5 passos cada
01

Abra o VSCode

Crie um arquivo chamado exercicio.html em qualquer pasta do seu computador. Esse será o seu laboratório.

02

Cole o esqueleto base

Cole esta estrutura uma vez: <script> … </script>. Dentro dela vai o código de cada exercício, um por vez.

03

Desenhe primeiro no Draw.io

Abra app.diagrams.net e desenhe o fluxograma seguindo os passos sugeridos. Só depois escreva o código.

04

Rode no navegador

Duplo clique no arquivo HTML ou use a extensão Live Server. As caixas prompt() e alert() vão aparecer.
