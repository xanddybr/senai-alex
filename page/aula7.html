Nível Básico

Fixando Vetores e Condicionais

01
O Array de Convidados
Objetivo
Criar um vetor simples e acessar suas posições.
Enunciado
Crie um vetor chamado convidados contendo 4 nomes. Use o console.log para exibir apenas o segundo e o quarto nome da lista. Depois, use o console.table(convidados) para ver a estrutura completa no console.
Dica

Lembre-se que em JavaScript os índices de um array começam em 0. O segundo elemento está na posição [1] e o quarto em [3].

02
Validador de Acesso — if/else com alert
Objetivo
Utilizar condicionais para retornar respostas booleanas via alerta.
Enunciado
Peça para o usuário digitar uma senha através de um prompt. Se a senha for igual a "1234", exiba um alert(true). Caso contrário, exiba um alert(false).
Dica

Use o operador de comparação === (triplo igual) para comparar strings com segurança. O == pode causar conversões de tipo inesperadas.

03
O Menu de Cores — for
Objetivo
Iterar sobre um vetor utilizando o laço for.
Enunciado
Crie um vetor chamado cores com as strings "Azul", "Verde", "Vermelho". Utilize um laço for para percorrer esse vetor e exibir cada uma das cores em um alert() individual.
Dica

Use cores.length como condição do for para que o laço se adapte automaticamente ao tamanho do vetor, sem precisar digitar o número de elementos na mão.

Nível Intermediário

Laços de Repetição Dinâmicos

04
Tabuada Dinâmica com for
Objetivo
Criar lógica aritmética dentro de um laço determinado.
Enunciado
Peça ao usuário um número de 1 a 10 via prompt. Utilize um laço for para calcular e exibir no console.log a tabuada desse número (de 1 a 10).
Saída esperada (se o usuário digitar 5)
console
5 x 1  = 5
5 x 2  = 10
5 x 3  = 15
... (até 5 x 10 = 50)
Dica

O retorno do prompt é uma string. Converta o valor para número com Number() ou parseInt() antes de usar em cálculos.

05
O Somador Infinito — while
Objetivo
Usar o while com uma condição de parada controlada pelo usuário.
Enunciado
Crie uma variável total = 0. Use um laço while que pede para o usuário digitar um número. Vá somando esses números na variável total. O loop só deve parar se o usuário digitar 0. No final, mostre o resultado com um alert.
Dica

Lembre de converter o valor do prompt para número antes de comparar com 0. A comparação "0" === 0 retorna false porque os tipos são diferentes.

06
Cadastro Obrigatório — do while
Objetivo
Garantir que o bloco rode pelo menos uma vez usando do while.
Enunciado
Crie um programa que pede a confirmação do e-mail do usuário. O do while deve continuar pedindo o e-mail enquanto o campo estiver vazio ou o texto digitado não contiver o caractere "@". Quando for um e-mail válido, exiba alert("E-mail salvo com sucesso!").
Dica

Use o método .includes("@") para verificar se a string contém o símbolo de arroba. Para checar se está vazio, teste se o valor é "" ou null (quando o usuário cancela o prompt).

Nível Avançado

Trabalhando com Matrizes (Arrays Bidimensionais)

07
A Batalha Naval Simplificada
Objetivo
Localizar elementos dentro de uma matriz estática usando índices.
Enunciado
Crie uma matriz 3×3 representando um mapa de água ("~") e navios ("X"). Peça para o usuário digitar a linha (0 a 2) e a coluna (0 a 2) via prompt. Se na posição escolhida tiver um "X", dê um alert(true). Se tiver "~", dê um alert(false).
Matriz inicial
javascript
const mapa = [
  ["~", "~", "X"],   // linha 0
  ["~", "X", "~"],   // linha 1
  ["X", "~", "~"]    // linha 2
];
Dica

Para acessar um elemento da matriz use mapa[linha][coluna]. Não esqueça de converter os valores do prompt para número antes de usar como índice.

08
O Diário de Notas
Objetivo
Alimentar uma matriz dinamicamente usando laços e condicionais.
Enunciado
Crie um vetor vazio chamado boletim. Utilizando um laço for que rode 3 vezes, peça ao aluno o seu Nome e a sua Nota final. Insira essas duas informações como uma linha dentro da matriz boletim usando .push([nome, nota]). No final, exiba a matriz usando console.table(boletim).
Estrutura esperada no console
console.table
// boletim após 3 cadastros:
[
  ["Ana",    8.5],
  ["Bruno",  7.0],
  ["Carla",  9.2]
]
09
Validador de Matriz — Campos Vazios
Objetivo
Validar dados inseridos em uma matriz e retornar alertas baseados no sucesso do cadastro.
Enunciado
Modifique o exercício anterior (Exercício 8). Durante o laço, se o aluno deixar o Nome ou a Nota em branco, o programa deve dar um alert(false) e não adicionar os dados na matriz. Se ambos estiverem preenchidos, dá um alert(true) e adiciona no boletim.
Dica

Use if (nome === "" || nota === "") para detectar campos vazios. Lembre que o prompt retorna null se o usuário clicar em Cancelar — também vale validar esse caso.

10
Desafio Master — O Carrinho de Compras
Objetivo
Unificar matrizes, acúmulo de valores e condição de saída com string.
Enunciado
Crie uma matriz vazia chamada carrinho. Usando a estrutura que preferir (while ou do while), crie um loop que pede o Nome do Produto e o Preço:
  • Se o usuário digitar "fim" no nome do produto, o programa para.
  • Caso contrário, adiciona o produto e o preço na matriz carrinho.
  • Ao encerrar, exiba a tabela final com console.table(carrinho).
  • Use um laço para somar o valor total de todos os produtos e exibir o total no final.
Estrutura esperada
console.table
// carrinho após as entradas:
[
  ["Notebook",  3500],
  ["Mouse",      89.9],
  ["Teclado",    150]
]
// Total: R$ 3739.90
Dica

Para somar os preços, percorra o carrinho com um for e acumule carrinho[i][1] (o índice 1 é onde está o preço). Converta cada preço para número com parseFloat() para aceitar valores com casas decimais.
