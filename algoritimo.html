30 problemas para pseudocódigo, fluxograma e console

Cada problema deve ser resolvido três vezes: primeiro como pseudocódigo no VisuAlg, depois como fluxograma no Draw.io, e por fim como código JavaScript testado no console do Chrome (F12). Mesma lógica, três formas de pensar.

Total de problemas 30 · 3 formas de resolver

Como resolver cada problema

Para cada um dos 30 problemas, siga estes 4 passos na ordem.

01

Pseudocódigo no VisuAlg

Abra o VisuAlg e escreva o algoritmo com Var, Inicio, leia, escreva e fimalgoritmo. Pressione F9 para executar.

02

Fluxograma no Draw.io

Abra app.diagrams.net e desenhe usando elipse (início/fim), paralelogramo (leia/escreva), retângulo (cálculo) e losango (decisão).

03

Console do Chrome

Abra o Chrome, pressione F12, vá na aba Console. Use prompt() para entrada e console.log() para saída.

04

Teste sempre

Rode com pelo menos 3 entradas diferentes para cada problema. Inclua casos extremos (zero, valor mínimo, valor máximo).
